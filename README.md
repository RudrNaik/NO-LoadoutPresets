# Loadout Presets

A BepInEx 5 mono mod for the game Nuclear Option.
Adds a new Window to the loadout select menu that allows you to easily select and customize preset loadouts.
Requirements: [BepInEx 5](https://github.com/BepInEx/BepInEx) [Install Instructions](https://docs.google.com/document/d/16aRWcrkt89YEn9_THwe9Fxo7AYPLE9SYSab_Y929dvE)

- `DEFAULT` loadout is volatile, it automatically saves the last loadout you used, other loadouts are changed manually in `edit` mode.
- `New` create a new preset with the specified name, the following characters are not allowed: `=` `\` `"` `'` `[` `]` `\r` `\n` `\t`.
- `Apply` update the selected preset with current loadout and name.
- `Delete` delete the selected preset, will request confirmation
